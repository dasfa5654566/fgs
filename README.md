# xray-replit

在Replit上部署vless+ws+tls。

**第一次进行完整配置，可能不算完善，仅供参考！** 目标在安全/稳定性和性能上均得到最优解。

## 注意

**请勿滥用！账号封禁风险自负。网络流量每月有100GB软上限。**

## 说明

使用Xray Core进行部署，同时使用Caddy 2进行HTTP伪装。
